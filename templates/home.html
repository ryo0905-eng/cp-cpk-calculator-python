Pick the right tool

If you want to know whether a process can meet specification limits, start with the Cp / Cpk calculator. If you already know there is a problem and want to test process settings, move to the DoE planner.

Suggested workflow

Review process capability first, decide whether the issue is spread or centering, then plan a structured experiment to test the settings that may improve the process.
